# FrontEnd DevStack

Based on **Bootstrap 3** with **BrowserSync** and **Kit** templates.

---

## How to start

1.  Download or clone repository
2.  run ```npm install```
3.  run ```bower install```
4.  run ```grunt``` and develop or ```grunt build``` for production
